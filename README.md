# hand-server

PHP-сервер для управления физической кистью(сжать/разжать) через HTTP-запросы.  

## Как запустить?
### 1. проверить наличие php
```bash
php -v
```
если нет, то установить.

### 2. Запуск сервера
В папке с проектом выполнить:
```bash
php -S localhost:5000
```
Для доступа из локальной сети заменить `localhost` на `0.0.0.0`:
```bash
php -S 0.0.0.0:5000
```

### 3. Использование
- **Сжать руку:**  
  `http://localhost:5000/hand.php?action=on` → ответ `CLOSED`

- **Разжать руку:**  
  `http://localhost:5000/hand.php?action=off` → ответ `OPENED`

## Остановка сервера
Нажмите `Ctrl+z` в терминале, где запущен PHP.
