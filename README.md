# Лампа

Лампа включается и выключается по ссылке.

## Как запустить

1. Запустите код в любом компиляторе Python (PyCharm, VS Code, IDLE) или прямо из терминала
2. После запуска в терминале появится надпись: `Сервер запущен: http://localhost:8080`
3. Откройте браузер и перейдите по этой ссылке

## Как управлять лампой

Лампа включается и выключается только сменой адреса в браузере:

- **http://localhost:8080/lamp_on** — включить 
- **http://localhost:8080/lamp_off** — выключить 


## Как выключить

Остановите код в компиляторе или нажмите **Ctrl+C** в терминале.
